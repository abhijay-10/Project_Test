# NLP, Bag of Words, etc..

## Project Overview
This project is an end-to-end NLP system that performs multiple tasks including:
- Chat Response Generation (using local LLM and Chat Models like mistral:latest)
- Text Summarization
- Sentiment Analysis

The system integrates modern AI technologies like Transformers, FastAPI, and Ollama (Mistral) to build a scalable and production-ready application.
